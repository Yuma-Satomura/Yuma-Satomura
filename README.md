### 👤 自己紹介

Web・モバイルの両方でプロダクトを開発しているエンジニアです。
アイデアを形にすることが好きで、**業務課題を解決するシステムを、企画から設計・実装・リリースまで一貫して手がける**ことを得意としています。

- 🧩 **得意領域**
  個人事業主向けの会計・帳簿 SaaS や、工事業向けの案件・工程管理システムなど、**現場の実務に根ざした業務アプリケーション**の開発。Next.js + Supabase を中心としたモダンな Web 開発から、Flutter によるクロスプラットフォームなモバイルアプリまで幅広く対応します。

- 💼 **開発経験**
  フロントエンド（Next.js / React / Astro）、バックエンド・DB（Supabase / PostgreSQL）、決済（Stripe）、モバイル（Flutter / Dart）まで、**個人でフルスタックに開発・運用**。法制度（電子帳簿保存法など）への対応や、実際に課金が走る本番サービスの構築・運用も経験しています。

- 🤖 **AI ツールの活用**
  Claude Code をはじめとする AI コーディングツールを日常の開発ワークフローに取り込み、**設計・実装・リファクタリング・デバッグを高速化**。AI を「使う側」だけでなく、AI を活用した学習プラットフォームや AI プロダクト（英会話・アシスタント系）を**自ら作る側**としても取り組んでいます。
